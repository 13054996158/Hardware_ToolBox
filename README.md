# Hardware_ToolBox

#### 卡片式布局电脑硬件工具箱

程序采用卡片式布局

提供了检测电脑硬件的详细详细

支持动态监测系统各项详细，如占用率，硬件频率

增加常用软件，如CPU-Z，GPU-Z，ADID64等工具
